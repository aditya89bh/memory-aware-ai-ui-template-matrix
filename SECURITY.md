# Security Policy

This repository primarily contains documentation and design artifacts. It does not ship runtime code or production software.

## Reporting concerns

If you find a security, privacy, or safety concern in the repository content, please open a private report if available or contact the maintainer directly.

Examples of relevant concerns:

- Guidance that could encourage unsafe handling of sensitive memory
- Broken documentation that misstates privacy or forgetting behavior
- Accidental inclusion of private information
- Links to unsafe or unrelated resources

## Scope

In scope:

- Repository documentation
- Templates, worksheets, examples, and review tools
- Contribution and release process files

Out of scope:

- Third-party products that use this repository
- Implementations derived from these templates
- General product security issues unrelated to this repository

## Response expectations

Maintainers should review credible reports, correct content when appropriate, and document significant changes in the changelog.
