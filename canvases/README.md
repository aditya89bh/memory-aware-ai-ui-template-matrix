# Canvases

Canvases are workshop-ready artifacts for structuring memory design decisions.

Use them when a team needs to align on memory governance, audit findings, consent, correction, forgetting, or overall product strategy.

## Canvases

- [Memory Governance Canvas](memory-governance-canvas.md)
- [Memory Audit Canvas](memory-audit-canvas.md)
- [Memory Consent Canvas](memory-consent-canvas.md)
- [Memory Correction Canvas](memory-correction-canvas.md)
- [Memory Forgetting Canvas](memory-forgetting-canvas.md)
- [AI Memory Product Canvas](ai-memory-product-canvas.md)

## Facilitation guidance

1. Pick one canvas for one product flow.
2. Fill it with a cross-functional group.
3. Capture decision outputs clearly.
4. Convert unresolved areas into issues, research questions, or design tasks.
