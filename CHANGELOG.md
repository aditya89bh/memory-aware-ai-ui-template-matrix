# Changelog

All notable changes to this repository will be documented here.

## v1.0.0 - Stable public release

### Added

- Complete memory-aware template matrix
- Applied examples across multiple product categories
- Review rubrics, worksheets, canvases, and visual frameworks
- AI product memory pattern guides
- Research methods, usability test plan, evaluation metrics, and testing scenarios
- Publication, release, support, citation, security, and contribution readiness files

### Status

This is the first stable public release. The repository is ready to use for product reviews, design critiques, workshops, consulting engagements, and research planning.

## v0.1.0 - Initial public foundation

### Added

- Repository vision, taxonomy, and memory design principles
- Template schema and matrix index
- Template families for capture, inspection, editing, correction, forgetting, scope, permissions, provenance, and confidence
- Applied examples across productivity, collaboration, support, education, robotics, research, and creative tools
- Review tools including risk rubric, scorecard, lifecycle rubric, workshop guide, stakeholder questions, pattern combinations, and anti-patterns
- Worksheets for product and design review
- Contribution guidance, issue templates, and pull request template
