# Worksheets

These worksheets are designed for product reviews, design critiques, workshops, and launch readiness checks.

They are intentionally plain Markdown so teams can copy them into planning docs, issue trackers, research notes, or design review artifacts.

- [Memory Flow Audit](memory-flow-audit.md)
- [Memory Risk Assessment](memory-risk-assessment.md)
- [Template Selection Worksheet](template-selection-worksheet.md)
- [Stakeholder Review Worksheet](stakeholder-review-worksheet.md)
- [Memory Lifecycle Planning](memory-lifecycle-planning.md)
- [Memory Correction Planning](memory-correction-planning.md)
- [Forgetting Controls Audit](forgetting-controls-audit.md)
- [Provenance and Confidence Audit](provenance-confidence-audit.md)

## Recommended use

Start with the memory flow audit, classify risk, select templates, then use specialized worksheets for correction, forgetting, lifecycle, and provenance gaps.
