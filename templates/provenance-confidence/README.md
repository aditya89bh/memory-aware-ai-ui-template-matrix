# Provenance and Confidence Templates

Provenance and confidence templates explain why a memory exists and how reliable it appears to be.

A memory-aware product should not present all memory as equally certain. A directly confirmed preference, a repeated behavior pattern, and a weak inference deserve different treatment.

## Design goal

Help users judge memory reliability and origin.

## Templates in this family

- [Why Remembered This](why-remembered-this.md)
- [Confidence and Source Indicator](confidence-and-source-indicator.md)

## Use this family when

- Users ask why something was remembered.
- Memory affects important outputs.
- Memory comes from inference or multiple sources.
- Recency and confidence matter.
- A user needs to challenge or verify memory.

## Design cautions

Do not overload users with technical scoring. Use human-readable labels and show evidence when it helps decision-making.
