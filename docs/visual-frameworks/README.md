# Visual Frameworks

Visual frameworks help teams teach, critique, and explain memory-aware product behavior without starting from a blank page.

Use these artifacts in design reviews, workshops, consulting sessions, and product strategy discussions. They are intentionally written in Markdown so they can be copied into slides, planning documents, design briefs, or workshop boards.

## How to use these frameworks

1. Choose one product flow where memory affects the user experience.
2. Pick the framework that matches the decision problem.
3. Walk through the framework with a cross-functional team.
4. Convert gaps into template selections, worksheet tasks, or launch blockers.
5. Document decisions in the relevant canvas or worksheet.

## Recommended pairings

- Lifecycle questions: use the lifecycle diagram and lifecycle planning worksheet.
- Consent questions: use the decision tree and consent canvas.
- Correction questions: use the correction loop and correction canvas.
- Forgetting questions: use the forgetting lifecycle and forgetting controls audit.
- Scope questions: use the scope map and stakeholder review worksheet.
