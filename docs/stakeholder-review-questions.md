# Stakeholder Review Questions

Use these questions when reviewing memory-aware features with cross-functional stakeholders.

## Product questions

- What product outcome does this memory support?
- What user problem becomes easier because memory exists?
- What would happen if the product did not remember this?
- Is the memory durable, temporary, or context-specific?
- What is the expected user benefit?

## Design questions

- Where does the user first learn that memory may be saved?
- Where can the user inspect memory later?
- How does the user know memory affected an output?
- How does the user correct wrong memory?
- How does the user forget memory?

## Research questions

- Do users understand the difference between current context and durable memory?
- Do users notice memory indicators?
- Do users trust correction receipts?
- Can users predict where memory will apply?
- What language do users naturally use for forgetting or correction?
