# Facilitation Scripts

Use these scripts to run memory design workshops and critiques.

## Opening script

"Today we are reviewing how memory affects this product flow. We are not reviewing implementation architecture. We are focusing on what the user sees, understands, controls, corrects, and forgets."

## Memory inventory prompt

"For each step in the flow, what might the product remember? Who benefits from that memory, and who could be confused or affected by it?"

## Risk classification prompt

"If this memory is wrong, stale, or visible in the wrong place, what is the worst realistic outcome?"

## Correction prompt

"Imagine the product used the wrong memory in this output. Where does the user notice it, and what is the fastest path to repair it?"

## Forgetting prompt

"When the user chooses forget, what exactly stops happening? Does the source remain? Could the memory be recreated?"

## Closing script

"We should leave with selected templates, unresolved risks, design changes, and owners. If users cannot inspect, correct, and forget memory that affects important outcomes, the flow is not ready."
