# Roadmap

This roadmap describes future directions after the v1.0.0 release.

## Near-term

- Add more industry examples for healthcare, finance, developer tools, sales, and operations.
- Add usability test report examples.
- Add more visual workshop boards.
- Expand pattern guides for multimodal, voice, and mobile products.

## Medium-term

- Add maturity model benchmarks.
- Add sample critique reports.
- Add more research protocols.
- Add translated terminology notes if contributors are available.

## Long-term

- Build a companion website or documentation site.
- Add downloadable design workshop packs.
- Maintain a public catalog of memory design anti-patterns and better alternatives.

## Contribution priorities

The most useful contributions are practical examples, tested worksheets, sharper edge cases, and clearer decision guidance.
