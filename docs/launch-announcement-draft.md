# Launch Announcement Draft

Use this draft as a starting point for a public launch post.

## Short version

Memory is becoming a core part of AI product experience, but most memory features are still hidden, hard to inspect, and difficult to correct.

Memory-Aware AI UI Template Matrix is a design reference for creating inspectable, editable, correctable, and forgettable memory experiences.

It includes templates, examples, worksheets, canvases, visual frameworks, review tools, and product pattern guides for teams designing transparent and controllable memory.

## Longer version

AI products increasingly remember preferences, projects, decisions, behavior, and context. That memory shapes outputs, recommendations, automation, and collaboration. If users cannot see or correct it, trust breaks.

This repository treats memory as a first-class design material. It helps product teams answer: What should be remembered? Where should it apply? How can users inspect it? How do they fix it? How do they forget it?

Start with the 30-minute usage guide, then use the template matrix, worksheets, and canvases to review one product flow.
