# Contribution Roadmap

This roadmap outlines useful ways to expand the repository.

## Near-term contributions

- More applied examples for healthcare, finance, developer tools, sales, and operations
- More template variations for mobile, voice, and collaborative interfaces
- More anti-patterns with corrective patterns
- More worksheets for research planning and usability testing

## Medium-term contributions

- Memory design evaluation methods
- Example critique reports
- Workshop facilitation kits
- Glossary refinements
- Visual diagrams that explain memory lifecycle and scope

## Contribution quality bar

A contribution should help a product team make a better decision. It should be concrete, reusable, and clear about user controls, trust risks, and edge cases.

## Good first contributions

- Add one realistic example flow
- Improve one template with sharper edge cases
- Add one anti-pattern and a better alternative
- Test a worksheet in a real design review and submit improvements
