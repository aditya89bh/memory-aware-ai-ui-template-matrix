# Publication Checklist

Use this checklist before sharing the repository publicly.

## Content readiness

- README explains the repository in under 60 seconds.
- Core navigation links are visible near the top.
- Template matrix, examples, worksheets, canvases, and pattern guides are linked.
- Terminology matches the glossary and taxonomy.
- No placeholder or filler pages remain.

## Contribution readiness

- Contributing guide exists.
- Issue templates exist.
- Pull request template exists.
- Code of conduct exists.
- Security policy exists.

## Release readiness

- Changelog includes the release.
- Release checklist is complete.
- Versioning guidance exists.
- Final quality audit is published.
- Relative links validate.

## Positioning readiness

- Repository positioning is clear.
- Audience is clear.
- Scope limits are stated.
- Future roadmap is available.
