# Pattern Combinations

Individual templates become more useful when combined into complete memory workflows.

## Basic preference workflow

Use for low-risk personal preferences.

1. [Inferred Preference Confirmation](../templates/memory-capture/inferred-preference-confirmation.md)
2. [Memory Drawer](../templates/memory-inspection/memory-drawer.md)
3. [Inline Memory Edit](../templates/memory-editing/inline-memory-edit.md)
4. [Forget Single Memory](../templates/memory-forgetting/forget-single-memory.md)

Best for summary length, view preferences, tone preferences, and routine formatting choices.

## Shared workspace workflow

Use when memory affects multiple people.

1. [Remember Across Workspace](../templates/memory-scope-permissions/remember-across-workspace.md)
2. [Source-Linked Memory Card](../templates/memory-inspection/source-linked-memory-card.md)
3. [Conflicting Memory Resolver](../templates/memory-correction/conflicting-memory-resolver.md)
4. [Memory Expiry Control](../templates/memory-forgetting/memory-expiry-control.md)

Best for team norms, project constraints, account facts, and shared delivery standards.

## Sensitive memory workflow

Use when memory touches personal, confidential, or high-impact context.

1. [Sensitive Memory Consent Gate](../templates/memory-scope-permissions/sensitive-memory-consent-gate.md)
2. [Contextual Memory Boundary](../templates/memory-capture/contextual-memory-boundary.md)
3. [Why Remembered This](../templates/provenance-confidence/why-remembered-this.md)
4. [Forget Single Memory](../templates/memory-forgetting/forget-single-memory.md)

Best for health-related constraints, employment context, financial goals, private customer details, and confidential business information.

## Research and evidence workflow

Use when memory must distinguish evidence, interpretation, and uncertainty.

1. [Explicit Memory Save Prompt](../templates/memory-capture/explicit-memory-save-prompt.md)
2. [Source-Linked Memory Card](../templates/memory-inspection/source-linked-memory-card.md)
3. [Confidence and Source Indicator](../templates/provenance-confidence/confidence-and-source-indicator.md)
4. [Remembered Facts Timeline](../templates/memory-inspection/remembered-facts-timeline.md)

Best for research synthesis, customer discovery, policy work, and strategic planning.
