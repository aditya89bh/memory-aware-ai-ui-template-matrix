# Support

This repository is maintained as an open design reference.

## Getting help

Use GitHub issues for:

- Template requests
- Documentation improvements
- Example submissions
- Anti-pattern submissions
- Questions about how to apply the matrix

Before opening an issue, check:

- [FAQ](faq.md)
- [Template Selection Guide](template-selection-guide.md)
- [How to Use This Repository in 30 Minutes](how-to-use-in-30-minutes.md)
- [Worksheets](../worksheets/README.md)

## Sponsorship and funding

No funding configuration is included at this time. If support options become relevant later, they should be documented transparently and kept separate from contribution quality expectations.
