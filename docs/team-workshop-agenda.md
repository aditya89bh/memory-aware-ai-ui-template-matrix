# Team Workshop Agenda

This agenda supports a 90-minute team workshop on a memory-aware product flow.

## Before the workshop

- Pick one product flow.
- Bring screenshots, prototype links, or journey maps.
- Ask participants to review the [Taxonomy](taxonomy.md) and [Template Matrix Index](../templates/matrix-index.md).

## Agenda

| Time | Activity | Output |
|---|---|---|
| 0-10 min | Define the flow and user goal | Shared scope |
| 10-25 min | Inventory memory objects | Memory list |
| 25-40 min | Classify risk and scope | Risk table |
| 40-60 min | Select templates | Pattern set |
| 60-75 min | Walk through correction and forgetting | Recovery paths |
| 75-90 min | Score and assign next steps | Action plan |

## After the workshop

Document selected templates, unresolved questions, owners, and launch blockers.
